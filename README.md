# 🌤 Weather App

A command-line weather application built in Python that fetches real-time weather data using the OpenWeatherMap API.

## Features

- Current weather by city
- Temperature
- Feels-like temperature
- Humidity
- Wind speed
- Atmospheric pressure
- Weather description
- Error handling for invalid cities

## Technologies Used

- Python
- Requests
- REST API
- JSON

## Installation

```bash
pip install -r requirements.txt
```

Add your API key to `config.py` and run:

```bash
python main.py
```
